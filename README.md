# College-Website
📌 Description

A simple College Website project built using HTML, CSS, and C. This project demonstrates a basic frontend design for a college website along with a C program to manage student data storage.

📖 About the Project

This project is designed to showcase a basic structure of a college website. It includes a homepage layout and a backend C program that allows users to input student details and store them in a file.

🚀 Features
Responsive homepage design
Navigation bar (Home, About, Courses, Contact)
Clean and simple UI
Student data entry using C program
File handling in C (stores data in data.txt)
🛠️ Technologies Used
HTML
CSS
C Programming
